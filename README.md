# Vite Vue Nuxt UI Starter

Minimal starter project with:

- Vite
- Vue 3 + TypeScript
- Tailwind CSS 4
- Nuxt UI 4

## Commands

```sh
npm install
npm run dev
```

```sh
npm run build
npm run preview
```

## Structure

- `src/main.ts`: app bootstrap
- `src/App.vue`: minimal starter screen
- `src/assets/style.css`: Tailwind and Nuxt UI imports plus base styling
